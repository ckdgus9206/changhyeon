# Github 사용법
> 2017.09.16 멀티캠퍼스 멋사 멘토링
---
## 1. Github 시작하기
- `git init`
- git을 시작한다
##현재 상태 확인
- `git status`
## 3. 파일추가
- `git add [파일명]` : 파일 하나 추가
- `git add . ` : 모든 파일을 추가
## 4. 저장하기
- `git commit -m "저장할 이름(메시지)"` : 저장 / 커밋
## 5. 저장기록 확인
- `git log`
## 6. 원격저장소(Github)에 올리기
- Github에 Repository 생성 하기
- `git remote add [원격저장소 주소]` : Repository와 프로젝트 연결
- `git push -u origin master` : 원격저장소에 코드 저장
